# \<yara-editor>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i yara-editor
```

## Usage
```html
<script type="module">
  import 'yara-editor/yara-editor.js';
</script>

<yara-editor></yara-editor>
```
